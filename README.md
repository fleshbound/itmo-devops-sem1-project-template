# Финальный проект 1 семестра

REST API сервис для загрузки и выгрузки данных о ценах.

## Требования к системе

Требования (рекомендуется):

- ОС - Ubuntu Ubuntu 24.04.3 LTS;
- процессор - 13th Gen Intel® Core™ i7-1360P × 16;
- объем оперативной памяти - 16 ГБ;
- наличие Go 1.24.0;
- развернутая БД PostgreSQL с пользователем validator

## Установка и запуск

Для локальной установки приложения требуется выполнить следующие шаги:

1. Скопировать репозиторий: `git clone git@github.com:fleshbound/itmo-devops-sem1-project-template.git`
2. Выполнить подготовку окружения (замените данные пользователя при необходимости в соответствии с состоянием вашей БД): `./scripts/prepare.sh`
3. Создать файл `.env`, скопировав `.env.example`, изменить значения переменных на свои
4. Запустить приложение: `./scripts/run.sh`

## Тестирование

Директория `sample_data` - это пример директории, которая является разархивированной версией файла `sample_data.zip`.

Приложение проходит тесты с помощью запуска в корне проекта следующей команды:

```bash
./scripts/tests.sh 1
```
